# HR-ANALYTICS-DASHBOARD
POWER BI HR DASHBOARD

##Overview
This project is an HR Analytics Dashboard built using Power BI to analyze employee data and generate insights on workforce trends.

## 🔧 Features
- KPI Cards: Total Employees, Average Age, Average Monthly Hours, Average Years at Company
- Slicers: Job Role, Education Level, Attrition Risk Level
- Visuals: Bar Chart, Scatter Plot, Pie Chart

## 📈 Insights
- Employee distribution across job roles
- Relationship between number of projects and working hours
- Gender-based employee distribution

## 🛠 Tools Used
- Power BI
- Data Cleaning & Modeling
